# 🖼️ Image Gallery using HTML, CSS & JavaScript

A modern and responsive **Image Gallery Web Application** created using **HTML**, **CSS**, and **JavaScript**.  
This project allows users to view images in a stylish gallery layout with lightbox preview, navigation controls, hover effects, and category filtering.

---

## 🚀 Features

### 📸 Gallery Functionalities
- 🖼️ Responsive Image Grid Layout
- 🔍 Lightbox Image Preview
- ⏭️ Next Image Navigation
- ⏮️ Previous Image Navigation
- ✨ Smooth Hover Effects
- 🎨 CSS Transitions & Animations

### 📱 Responsive Design
- Mobile Friendly Layout
- Adaptive Grid System
- Optimized for Different Screen Sizes

### ⭐ Bonus Features
- 🏷️ Image Categories / Filters
- 🎞️ Smooth Image Transitions
- 🖱️ Interactive UI Effects

---

## 🛠️ Technologies Used

- **HTML5** → Structure of the gallery
- **CSS3** → Styling, animations, and responsive design
- **JavaScript (ES6)** → Lightbox and navigation functionality
